Embedded Widgets for Notion.

“Notion is a freemium productivity and note-taking web application developed by Notion Labs Inc. It offers organizational tools including task management, project tracking, to-do lists, bookmarking, and more.”

These free HTML widgets, designed by me, are easily embedded into your Notion pages. Along with using the provided embedded link, you can save the original source code to preserve and/or edit the widget yourself.

These widgets take love and time to make. Please consider supporting me by sharing. ♡
